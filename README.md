# 🌸 Happy Mother's Day — Amma Website

## Folder Structure

```
mothers-day/
├── index.html          ← Main website (open this in any browser)
├── images/             ← All 14 photos
│   ├── img1.jpg
│   ├── img2.jpg
│   └── ... (img3 to img14)
├── audio/              ← Place your background music here
│   └── amma.mp3        ← ADD your own music file here!
└── README.md
```

## 🎵 Adding Background Music

1. Find a beautiful Tamil melody (e.g., "Amma Endrazhaikatha" or any soft instrumental)
2. Convert/download it as **amma.mp3**
3. Place it inside the `audio/` folder
4. The 🔇 button (top right) will play/pause it

> Without the audio file, everything else works perfectly — the music button simply won't play anything.

## 🌐 How to Open

- Double-click `index.html` to open in your browser
- Or share the entire folder via Google Drive / WeTransfer
- Or host on GitHub Pages / Netlify for a shareable link
